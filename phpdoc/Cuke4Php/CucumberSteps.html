Class CucumberSteps

Description
Description | Descendents | Vars (details) | Methods (details)

Base class that all cucumber steps should derive from

Each step gets a reference to the global variables array provided by the scenario. This allows steps to save and maintain state over the course of a scenario run.

This class also inherits PHPUnit assertions, so those can all be used within the context of a step. Data providers probably don't work, and really should not be used in any case.
